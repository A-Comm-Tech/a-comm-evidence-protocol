# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct. The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Our standards, in short

We are building a neutral standard for a competitive industry. Participants are expected to:

- Argue **technically**, with citations to spec sections and primary sources.
- Represent your perspective honestly (actor class matters; affiliation disclosure is encouraged, never required).
- Refrain from vendor marketing, competitive disparagement, and personal attacks — the same neutrality rules that bind the spec's editors bind discussion here.
- Respect coordinated-disclosure norms for security findings (see [SECURITY.md](SECURITY.md)).

## Enforcement

Instances of unacceptable behavior may be reported to **conduct@a-comm.ai**. Reports are reviewed by the maintainers; consequences follow the Contributor Covenant's enforcement guidelines (correction → warning → temporary ban → permanent ban). Maintainers who violate the code are subject to the same process.
